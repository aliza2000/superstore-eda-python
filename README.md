# Superstore Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis on a Superstore dataset to understand
sales, profit, discount, and customer behavior using Python.

## Dataset
The dataset contains order-level information such as sales, profit, discount, quantity,
category, region, and order dates.
Original data source: Kaggle.
The cleaned dataset is provided as superstore-eda-cleaned.csv.

## Analysis Performed
- Data cleaning and preprocessing
- Univariate analysis
- Bivariate analysis
- Category and region-wise analysis
- Monthly sales and profit trends

## Key Insights
- Technology category is the most profitable
- High discounts often reduce profit
- West region generates the highest profit
- Most orders have low quantity values

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
